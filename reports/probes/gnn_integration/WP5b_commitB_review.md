## Findings (all LOW / informational — no blockers)

- **F1 (LOW, export, operational).** Candidate selection is by FILE MTIME (`select_candidate_paths`). mtime is not stable across clone/rsync/bulk-copy, so the pre-cutoff cohort could differ per host. Mitigated fail-loud: any resulting SET change flips the games-manifest sha ⇒ hard-fail (never a silent wrong export). Compounds Deviation 1 (no canonical sha registered — launch-prep must mint it). Recommend the launch-prep step mint + record the sha on the production box and note the mtime dependency.
- **F2 (LOW, export, boundary blind spot).** Truncation proxy `positions[-1].ply < expected_last_ply` misses an illegal `apply_move` at exactly the last in-window ply (`expected_last_ply`): that row (illegal-move target) would be pushed + persisted, caught LOUD only at training-sample `mass_drop_check`, not at export. Very narrow (human games end legally; corruption at the boundary ply is rare) and fail-loud, not silent. Optional hardening: gate the built row on the move being a legal cell before push.
- **F3 (INFO, trainer).** `collate_graph_batch` call relies on default `trunk_size=19`/`win_length=6`/feat dims rather than reading `spec.trunk_size`. Correct for the sole graph encoding `gnn_axis_v1`; a future second graph encoding with different geometry would silently mis-collate. Consider threading `spec` fields at the call site if/when a second graph encoding lands.
- **F4 (INFO, recency).** A de-duped recent-slice index is replaced by a full-ring `weighted_sample_one` draw, so for a game_id-TAGGED buffer dedup could dilute the recent fraction. Only reachable for tagged buffers (corpus mixing = DEFERRED); run4 self-play rows are `-1` ⇒ never fires. Spec-accepted approximation (delta §4).
- **F5 (INFO, monitoring).** Graph `train_step` structlog event adds `n_graphs`/`n_legal_nodes`/`fp16_scale` (not the delta doc's suggested `mean_legal_nodes`/`mean_edges`/`off_window_target_frac`, which were marked OPTIONAL). No panel consumes them ⇒ no producer-manifest violation (the law gates PANELS). Cosmetic.
